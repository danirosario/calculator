
# Calculadora Simples em C
Uma calculadora simples e interativa desenvolvida em C que permite realizar operações matemáticas de forma contínua, mantendo o resultado anterior para novos cálculos.




## Funcionalidades
A calculadora suporta as seguintes operações:

- **Operações básicas:** adição (+), subtração (-), multiplicação (*), divisão (/)
- **Potenciação:** eleva um número a uma potência (^)
- **Cálculos contínuos:** o resultado de cada operação se torna o valor inicial da próxima
- **Limpeza de memória:** reinicia os cálculos a qualquer momento (c)
- **Interface limpa:** limpa a tela automaticamente entre operações

## Como utilizar

#### 1. Digite o número inicial quando solicitado
#### 2. Escolha uma operação:

- ##### **+** para adição
- ##### **-** para subtração
- ##### __*__ para multiplicação
- ##### **/** para divisão
- ##### **^** para potenciação
- ##### **c** para limpar e começar novo cálculo
- ##### **q** para sair

#### 3. Digite o próximo número

#### 4. O resultado será exibido e você pode continuar calculando!


##  Proteções Implementadas

- ##### **Divisão por zero:** o programa detecta e exibe erro sem travar
- ##### **Operadores inválidos:**  mensagem de erro para operadores não reconhecidos
- ##### **Loop contínuo:** permite múltiplas operações sem reiniciar o programa
